NGÔN NGỮ HTML

Hyper text Markup language

* Cấu trúc trang html :

< html >
< head >
< title > tiêu đề trang < /title >
< body bgcolor="#445566" text="#aacc88" >
nội dung hiển thị (tự tìm tài liệu tham khảo)
< /body >
< /html >

* Các thẻ lệnh :

< b > chữ đậm < /b >
< i > chữ nghiêng < /i >
< u > chữ gạch chân < /u >

* Liên kết :

< a href="URL" target="_target">
< img src="img/home.gif" alt="#" > < /a >
